Proyecto de Programación Estructurada - Equipo UmiZoomi
Objetivo
Desarrollar una solución funcional bajo el paradigma de programación estructurada utilizando el lenguaje Python, cumpliendo con los lineamientos de la actividad

Requisitos
Python 3.10 o versiones superiores.
Herramienta Git para el control de versiones.
Editor de código: Visual Studio Code (VS Code).
Instrucciones de ejecución
Clonar el repositorio en su máquina local: git clone [https://github.com/GerardoAlbertoPerez/Equipo_UmiZoomi]
Abrir la carpeta del proyecto en VS Code.
Ejecutar el archivo principal desde la terminal integrada: python src/main.py
